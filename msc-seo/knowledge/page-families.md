# Approved SEO Page Families

These are the page structures the AI agent is allowed to generate for MySupermarketCompare.com.

Pages must follow these structures to ensure SEO quality and compliance.

---

## Car Insurance Pages

/car-insurance/{city}

/car-insurance/{driver-type}

/car-insurance/{situation}

Examples:

/car-insurance/london
/car-insurance/manchester
/car-insurance/young-drivers
/car-insurance/new-drivers
/car-insurance/no-deposit

---

## Core Rule

The AI must only generate pages that follow these page family structures.

If a keyword does not match one of these structures, the page must not be created automatically.
