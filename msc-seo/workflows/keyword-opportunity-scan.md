# Keyword Opportunity Scan

This workflow helps identify new SEO opportunities for MySupermarketCompare.com.

The goal is to discover keywords that competitors rank for or that appear in search results but are not yet covered by MySupermarketCompare.

---

Step 1 — Identify target keyword clusters

Focus on high-intent insurance queries such as:

- insurance + city
- insurance + driver type
- insurance + pet breed
- insurance + travel destination
- insurance + vehicle type

---

Step 2 — Compare against existing MSC coverage

Check whether MySupermarketCompare already has a page that covers the topic.

If a page already exists, record it as **covered**.

If not, mark it as a **potential SEO opportunity**.

---

Step 3 — Check page family compliance

Ensure the keyword fits one of the approved page families defined in:

knowledge/page-families.md

If the keyword does not match a page family, do not suggest creating the page.

---

Step 4 — Evaluate usefulness

Ask:

Would this page help a user compare insurance quotes?

If the answer is yes, it can be recommended.

---

Step 5 — Generate recommendation

Provide a recommendation including:

- keyword opportunity
- proposed URL structure
- page family
- reason the page would be useful
