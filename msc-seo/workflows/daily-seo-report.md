# Daily SEO Report

This workflow generates a daily SEO intelligence report for MySupermarketCompare.com.

The purpose is to identify the most important SEO actions that should be taken to grow organic traffic.

---

Step 1 — Run keyword opportunity scan

Execute the workflow:

workflows/keyword-opportunity-scan.md

Collect the highest-value keyword opportunities.

---

Step 2 — Run competitor gap analysis

Execute the workflow:

workflows/competitor-gap-analysis.md

Identify competitor pages that MySupermarketCompare does not yet cover.

---

Step 3 — Prioritise opportunities

Rank opportunities based on:

• search intent (high comparison intent)
• traffic potential
• relevance to MSC verticals
• ability to guide users toward comparing quotes

---

Step 4 — Generate page recommendations

Provide a list of recommended new pages including:

• target keyword
• proposed page URL
• page family
• reason the page should be created

---

Step 5 — Identify existing page improvements

Suggest improvements for existing MSC pages such as:

• better internal linking
• FAQ additions
• additional comparison information
• improved SEO titles

---

Step 6 — Produce daily report

Generate a structured report including:

1. Top new SEO page opportunities
2. Competitor keyword gaps
3. Recommended page improvements
4. Internal linking opportunities
